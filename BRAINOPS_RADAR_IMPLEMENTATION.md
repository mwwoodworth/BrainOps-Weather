- Located below the radar map.
- Simulates proactive intelligence (e.g., "Heavy Rain Approaching in 23 mins").
- Color-coded severity indicators (Critical, Warning, Info).
- Styled with a glass-morphism background to float over the content.

### 4. Performance Optimizations
- **Hardware Acceleration:** Explicitly enabled in `AndroidManifest.xml`.
- **Tile Caching:** Implemented `RadarTileCache` (LRU Cache) to minimize network calls and speed up animation playback.
- **Efficient Composable:** `RadarCard` uses lightweight Compose structures to ensure main-thread smoothness.
